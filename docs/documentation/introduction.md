# Introduction

Bienvenue sur la documentation CUB Édimbourg produite en Markdown avec MkDocs.

## 🌐 Agence Édimbourg

Cette documentation regroupe l'ensemble des travaux réalisés dans le cadre du **BTS SIO SISR** pour l'agence fictive **CUB Édimbourg**.

### Comment utiliser cette documentation

- **Navigation** : Utilisez la barre latérale gauche pour accéder aux différentes sections.
- **Recherche** : Utilisez la barre de recherche en haut pour trouver rapidement une information.
- **Copier les commandes** : Chaque bloc de code dispose d'un bouton copier en haut à droite.

### Structure

- **Situations** : Documentation des situations professionnelles (Windows Server, Git, DHCP, AD, GPO)
- **Cybersécurité** : Pare-feu Stormshield, cryptographie, bastion PAM, filtrage, SIEM Wazuh
- **DNS** : Serveurs récursifs, autorité, esclave et délégation de zone
- **Configurations** : Fichiers de configuration des équipements réseau Cisco
